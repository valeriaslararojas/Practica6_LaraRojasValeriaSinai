## 🌌 Descripción

En esta práctica se desarrolla el **modelado de un sistema eléctrico tipo convertidor CD-CD**, analizando su comportamiento mediante ecuaciones y herramientas de simulación implementadas en Matlab, generando un sistema con las herramientas de Simscape y simulink.

El sistema a simular es un Convertidor de potencia CD-CD.

---

## 🔥 Objetivos

🌿 Comprender el funcionamiento de un **convertidor CD-CD**
🌊 Aplicar técnicas de **modelado matemático de sistemas eléctricos**
🌪️ Analizar el comportamiento del sistema mediante **simulación**
🔥 Interpretar la respuesta del sistema a partir del modelo obtenido
